



### 操作

1. 登录https://icomoon.io/

2. 点击 IconMoonApp

3. Import Icons 

4. 选择 Edit （笔状 icon）清掉颜色

5. 右下角 GenerateFont 

6. Preferences 调整参数

7. Download

8. 下载压缩包，解压后放入对应文件夹

9. 需要修改 style.scss 中对应 font 的路径

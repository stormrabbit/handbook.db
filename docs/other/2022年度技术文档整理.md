[js 中的File不能修改文件名的问题解决方案](https://www.jianshu.com/p/6ba0c0ac90b1)
> 这是一个现实中遇到的问题，即在使用上传接口时，接口参数只有 file 属性，没有 filename 属性。造成的结果便是，在上传文件时必须用这种办法才能处理文件名中的特殊字符。因此在上传接口中，除接收上传文件之外，还应接收文件名以方便处理。

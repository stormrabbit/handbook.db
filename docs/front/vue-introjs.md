## 使用心得

#### 安装&引入

1. 安装

```
npm install intro.js --save
npm i vue-introjs
```

2. 引入

- 代码

```
import introJs from 'intro.js'
import VueIntro from 'vue-introjs'
window.introJs = introJs
Vue.use(VueIntro);
```

- 样式

```
@import '~intro.js/introjs.css';
```
